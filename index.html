DOW  = Base OEW + Crew × 85 kg
TOF  = Ramp Fuel − Taxi Fuel
LDF  = TOF − Trip Fuel
ALW  = min(MSLW, PLLW)
─── ATW paths (ใช้ค่าน้อยสุด) ───
ATW₁ = ALW + Trip Fuel
ATW₂ = MSZFW + TOF
ATW₃ = min(MSTOW, PLTOW)
ATW  = min(ATW₁, ATW₂, ATW₃)
AZFW = ATW − TOF
MTL  = AZFW − DOW

SCR / GCR Message Format & Quick Reference ครบถ้วน
⚠ ทุก Time ในข้อความ SCR/GCR ต้องเป็น UTC เท่านั้น
SCR — New (N)
SCR <Message Type> S26 <ฤดูกาล: S=Summer W=Winter + ปี 2 หลัก> 19OCT <วันที่ส่งข้อความ DDMMM> BKK <IATA Airport Code ที่ขอ Slot> N 8K8XX 8K8YY 30MAR26OCT 1010101 000737 VTBS0100 0200VTBS FF SI ALL TIMES UTC. GI BRGDS/KITTIPHOP C.
N = New Request  |  8K8XX = Flight No.ขาเข้า  |  8K8YY = ขาออก
30MAR26OCT = Period Start/End  |  1010101 = วันบิน (จ-อ-พ-พฤ-ศ-ส-อา)
000737 = Seats(000)+IATA AC Type(737=B737)  |  VTBS0100 = Origin+ArrTime
0200VTBS = DepTime+Dest  |  FF = Service Type (All Cargo Sched.)
SCR — Change (C) / Revise (R)
SCR S26 19OCT BKK Slot เดิม (C = Change): C 8K8XX 8K8YY 30MAR26OCT 1010101 000737 VTBS0100 0200VTBS FF Slot ที่ขอใหม่ (R = Revise): R 8K8XX 8K8YY 30MAR26OCT 1010101 000737 VTBS0115 0215VTBS FF SI REQ 15 MIN LATER ARR/DEP. ALL TIMES UTC. GI BRGDS/KITTIPHOP C.
ใช้ C+R คู่กันเสมอ — C คือ Slot เดิม, R คือ Slot ที่ต้องการใหม่
SI = Supplementary Info  |  GI = General Info (ผู้ส่ง)
SCR — Delete (D)
SCR S26 19OCT BKK D 8K8XX 8K8YY 30MAR26OCT 1010101 000737 VTBS0100 0200VTBS FF SI CANCELLATION OF ENTIRE SERIES. ALL TIMES UTC. GI BRGDS/KITTIPHOP C.
ยกเลิก Slot ทั้ง Series  |  ใช้ข้อมูล Slot เดิมทุกตัว
GCR — General Aviation (ขาเดียว)
GCR <General Aviation Request> /HSKMA <ทะเบียนเครื่องบิน (เว้นวรรคหน้า)> VTBS <ICAO Airport Code> New Arrival Slot: N HSKMA 20OCT 010737 VHHH0430 D Change + AC Type Change: C HSKMA 20OCT 010737 VHHH0430 D R HSKMA 20OCT 010767 VHHH0430 D SI AIRCRAFT TYPE CHANGE. NO TIME CHANGE. GI BRGDS/KITTIPHOP C.
010737 = Seats(010)+ICAO Type(737)  |  VHHH0430 = ICAO Origin+ArrTime(UTC)
D = General Aviation Flight  |  010767 = เปลี่ยนเป็น B767

Planning Minimums
เลือก Plan A / Plan B / N/A แถวจะเปลี่ยนสีอัตโนมัติ | กด ✏ แก้ไขข้อมูล | กด + เพิ่มสนามบิน
Plan A — "กฎระบบเดี่ยว"
Single Runway / One Facility (ILS หรือ VOR)
คำนวณเกณฑ์อากาศจากระบบเดียว | ต้องบวกค่าความปลอดภัยเพิ่ม
DA/H + 400 ft  |  RVR/VIS + 1,500 m
Plan B — "กฎระบบคู่แยกอิสระ"
Two Separate Runways & Facilities
คำนวณทั้ง 2 ระบบแยกกัน → เลือกค่าสูงกว่า (Higher of the Two)
DA/H + 200 ft  |  RVR/VIS + 800 m
Circling Approach
MDA/H + 400 ft  |  VIS + 1,500 m
⚠ เงื่อนไขการเลือกสนามบินสำรองปลายทาง
• อากาศ Destination ดีกว่า Planning Minima (ETA ± 1 ชม.) → เลือก Alternate 1 แห่ง
• อากาศ Destination ต่ำกว่า Planning Minima หรือไม่มีข้อมูลอุตุนิยมวิทยาที่แน่ชัด → เลือก Alternate 2 แห่ง (ทั้งสองแห่งต้องผ่านเกณฑ์)
Plan A → ระบบเดี่ยว → บวก DA/H+400 / VIS+1,500 → อากาศต้องดีมากกว่าจึงผ่านเกณฑ์
Plan B → ระบบคู่แยกกัน → บวก DA/H+200 / VIS+800 → เกณฑ์ต่ำกว่า โอกาสผ่านสูงขึ้น

☁ Weather Planning Minimum
Departure
• ทัศนวิสัยต้องดีกว่าค่าขั้นต่ำรันเวย์นั้นๆ รวม Takeoff Performance

Destination (with Alternate)
• TAF (ETA ± 1 ชม.) ทัศนวิสัยดีกว่า Landing Minimum
• ถ้าพอดี/ต่ำกว่า → ต้องมี Alternate 2 แห่ง

Destination (No Alternate)
• Flight time ≤ 6 ชม. (TAF อาจ Error ถ้าเกิน)
• TAF (ETA ± 1 ชม.) VIS > 5–8 km
• Ceiling > 2,000 ft AGL
• ไม่มีรายงานอากาศต่ำกว่า Landing Minima
• 2 Runway แยกกัน + เครื่องช่วยลงจอดเป็นของตัวเอง
• NOTAM, ATC, RFFS พร้อม

Alternate
• TAF (ETA ± 1 ชม.) ทัศนวิสัยดีกว่า Alternate Minimum
• NOTAM เปิด + มีอุปกรณ์ช่วยเหลือ

🌟 RNAV vs RNP
RNAV
บินตาม Waypoints โดยตรง ไม่ต้อง VOR-to-VOR
ไม่มีระบบเตือนภัย นักบินต้องตรวจสอบเอง
เช่น RNAV 1 = ±1 NM
RNP
เหมือน RNAV + มี OPMA
(On-board Monitoring & Alerting)
ระบบเตือนอัตโนมัติ สามารถบินโค้งได้
เช่น RNP 0.3 = ±0.3 NM
📡 ILS Components
Localizer (LOC) — แนว ซ้าย-ขวา ให้ตรงกึ่งกลาง Runway (110–111.95 MHz)
Glide Slope (GS) — แนว สูง-ต่ำ มุมร่อนปกติ 3° (329.15–335.00 MHz)
Outer Marker (OM) — ห่าง ~7.2 km จาก Threshold เสียง 400 Hz ——
Middle Marker (MM) — ห่าง ~1.1 km เสียง 1300 Hz -.-.-
Inner Marker (IM) — CAT II/III ใกล้ Threshold เสียง 3000 Hz ···

💰 Contingency Fuel
ERA Required: ต้องมีสนามบินสำรองระหว่างทาง (En-Route Alternate) ที่เหมาะสม อากาศดี เปิดให้บริการ จึงใช้ Rule 3% ได้
5% Standard: Trip Fuel × 0.05 — ใช้ได้ทุกกรณี
3% with ERA: Trip Fuel × 0.03 — ประหยัดน้ำมัน/รับสินค้าเพิ่ม
